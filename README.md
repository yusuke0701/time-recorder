# Time Recorder

[![CircleCI](https://circleci.com/gh/yusuke0701/time-recorder.svg?style=svg)](https://circleci.com/gh/yusuke0701/time-recorder)

## 概要

時間を記録するツールです。

## 主な構成

- クライアント
  - Chrome 拡張ツール
  - Vue.js
- サーバー
  - Google Cloud Functions
  - Golang
- データベース
  - Google Cloud Datastore

- ポップアップを非表示にすると ID が消えるので、保存する場所を変更する
